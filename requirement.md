CIS223 網頁應用程式開發 CA / CIS226 進階網頁應用程式開發
分配（30%）
背景
貴公司受聘為網頁應用程式開發者，現在被要求建立一個網頁應用程式，協助機構進行線上調查。
該網頁應用程式需要具備以下功能：
•	應收集受訪者資料。
•	受訪者可以瀏覽網站。
•	受訪者可以在網路上回答所有問題。（至少有10頁，包含不同控制方式）
•	所有答案都應該儲存在 mysql 資料庫中。
學生應該在本地機器上架設伺服器，並只在本地機器上運行網頁。因此，學生應安裝網頁伺服器與資料庫伺服器。你可以選擇任何平台和框架來建立網站。
學生應使用 html、css 或/和 javascript 來設計自己的網站。 
學生也應該設計資料庫並建立表格。 
交付成果：
一.	完整網站
把根目錄壓縮後提交到 Moodle。
二.	資料庫 
將資料庫匯出為.sql檔案並提交到 moodle。
三.	使用手冊
用 Word 文件製作使用者手冊並提交給 Moodle。
